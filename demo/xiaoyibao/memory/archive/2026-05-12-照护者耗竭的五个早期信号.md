# 照护者耗竭的五个早期信号：在崩溃前先识别它

长期照护重病家人的人，往往是最后一个发现自己已经撑不住的人。根据 NCCN 痛苦管理指南，照护者的心理社会风险需要与患者同等对待。耗竭不是「不够爱」，而是一种可被识别、可被干预的状态。

## 1. 识别五个早期信号

1. **睡眠崩坏**：长期早醒或入睡困难，醒来后毫无恢复感。
2. **情绪钝化**：对原本在意的事失去反应，只剩机械地完成任务。
3. **易怒与内疚交替**：因小事爆发，事后又陷入强烈自责。
4. **躯体化**：头痛、心悸、肠胃紊乱，检查却查不出器质性问题。
5. **社交退缩**：拒绝一切与照护无关的联系，把自己彻底封闭。

## 2. 可执行的应对

1. **承认边界**：照护强度需要被拆分，不要把自己当成唯一的支撑点。
2. **建立求助清单**：列出可以分担的人和资源，哪怕只是帮忙买一次药。
3. **保留一个出口**：每天留出最小的、只属于自己的时间。
4. **必要时就医**：持续两周以上的功能受损，应寻求专业心理支持。

先稳住自己，才谈得上托举家庭。识别信号，是停止内耗的第一步。
